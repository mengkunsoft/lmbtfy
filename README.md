### 更新日志
-----

#### 2020/6/21
- 重构项目，风格与新版百度 UI 适配。 **注意：本版本修改了参数的获取方式，与旧版不兼容！如果你曾经使用了旧版，那么不要替换为新版，否则以前生成的链接都会失效！**
- 移除短网址模块
- 增加点击指示特效
- 增加“我是自己人”按钮，防止自己人被“带跑”

#### 2019/10/14
- 将网址缩短接口替换为搜狗接口，修复无法生成短网址的问题

#### 2019/4/27
- 修复因 CDN 失效导致无法使用的问题
- 重构部分代码，界面更美观

#### 2017/3/16
- 将界面修改为响应式，以适应移动端访问
